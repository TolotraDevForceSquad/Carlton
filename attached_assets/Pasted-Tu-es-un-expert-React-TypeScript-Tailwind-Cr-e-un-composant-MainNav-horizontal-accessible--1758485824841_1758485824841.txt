Tu es un expert React + TypeScript + Tailwind. Crée un composant <MainNav /> horizontal, accessible et responsive pour un site hôtel 5★ (dark UI). Contraintes et attentes:

STACK & STYLE
- React + TypeScript + Tailwind uniquement (pas de CSS externe, pas de lib de menu).
- Thème sombre (#0f1115 approx). Bouton "Réserver" style or (#f2c451), arrondi lg, hover/active.
- Icônes: utilise emoji natifs (🌐, ☰) ou lucide-react si dispo.
- Header sticky en haut, largeur max xl, padding x 4, y 2, shadow subtle.

COMPORTEMENT
- Desktop (≥1024px): 
  - Menu principal HORIZONTAL avec items alignés sur une seule ligne: Accueil, Séjour, Restauration, Bien-être, Événements, Découverte, Contact.
  - Les sections ayant des sous-liens ouvrent un sous-menu HORIZONTAL en dropdown (panel aligné sous l’item, fond #151821, border, shadow).
  - Navigation au clavier: Tab/Shift+Tab, Esc pour fermer, focus visible.
  - Hover et focus ouvrent le sous-menu; en dehors, il se ferme.
- Mobile (<1024px):
  - Icône ☰ à droite ouvre un drawer/overlay vertical plein écran (position fixed).
  - Dans le drawer, les sections avec sous-liens sont accordéons (chevron, ouverture/fermeture).
- À droite du header: 
  - Toggle langue (FR/EN) avec 🌐 + code langue.
  - Bouton "Réserver" (CTA principal).
  - Icône ☰ (mobile uniquement).
